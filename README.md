## About the Project

This is a simple Python program that calculates the total investment value of stocks. The user enters stock names and quantities, and the program uses fixed prices to compute the total.

## How It Works

* Stock prices are stored in a dictionary.
* The user enters the stock name and quantity.
* The program calculates the value.
* It shows the total investment.
* Optionally, the result can be saved to a file.

## Concepts Used

* Dictionary
* Input/Output
* Basic arithmetic
* File handling

## How to Run

1. Save the file as `stock_tracker.py`
2. Run:

   ```
   python stock_tracker.py
   ```
